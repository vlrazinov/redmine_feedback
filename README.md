# Redmine Feedback Plugin

Плагин для Redmine, позволяющий собирать оценку качества поддержки от пользователей непосредственно в интерфейсе задач.

## Автор
**Разинов Владислав**

## Функциональность

Плагин предоставляет следующий функционал:

1. **Автоматическое создание поля оценки**
   - При установке плагин автоматически создаёт собственное кастомное поле типа "Список" с названием "Оценка поддержки".
   - Поле содержит варианты оценок: "Отлично", "Хорошо", "Удовлетворительно", "Плохо".
   - Плагин не зависит от жёстко заданных ID полей, что обеспечивает корректную работу на любых стендах.

2. **Интерфейс оценки в задаче**
   - В карточке задачи (после описания или в боковой панели) отображается блок с запросом оценки качества поддержки.
   - Пользователь может выбрать одну из предложенных оценок и оставить комментарий (опционально).

3. **Сохранение результатов**
   - Выбранная оценка и комментарий сохраняются в созданное плагином кастомное поле задачи.
   - Данные доступны для построения отчётов и анализа качества поддержки.

4. **Безопасность и совместимость**
   - Плагин корректно работает в многопользовательской среде.
   - Не конфликтует с другими плагинами благодаря использованию собственного изолированного поля.
   - Совместим с Redmine 4.x и 5.x.

## Процедура установки

### Требования
- Redmine версии 4.0 или выше
- Доступ к командной строке сервера с правами пользователя, под которым запущен Redmine
- Установленные зависимости Ruby и Rails, соответствующие вашей версии Redmine

### Шаг 1: Клонирование или копирование плагина

Перейдите в директорию `plugins` вашего Redmine и клонируйте репозиторий плагина (или скопируйте файлы плагина):

```bash
cd /path/to/redmine/plugins
git clone <url_репозитория> redmine_feedback
# Или скопируйте файлы плагина вручную в папку redmine_feedback
```

Убедитесь, что папка с плагином называется `redmine_feedback`.

### Шаг 2: Установка зависимостей и миграция базы данных

Выполните команду для установки необходимых зависимостей (если есть) и запуска миграций:

```bash
cd /path/to/redmine
bundle install --without development test
bundle exec rake redmine:plugins:migrate RAILS_ENV=production
```

> **Важно:** На этом этапе плагин автоматически создаст в базе данных кастомное поле "Оценка поддержки".

### Шаг 3: Перезапуск Redmine

Перезапустите веб-сервер или приложение Redmine, чтобы изменения вступили в силу:

```bash
# Для systemd
sudo systemctl restart redmine

# Или для Passenger/Apache
sudo systemctl restart apache2

# Или для Puma/Unicorn в зависимости от вашей конфигурации
sudo systemctl restart puma
```

### Шаг 4: Проверка установки

1. Откройте любую задачу в Redmine.
2. Найдите блок "Оценка качества поддержки".
3. Попробуйте выставить оценку и сохранить.
4. Проверьте в администрировании -> настраиваемые поля, что поле "Оценка поддержки" создано и имеет тип "Список".

## Удаление плагина

Если потребуется удалить плагин, выполните следующие команды:

```bash
cd /path/to/redmine
bundle exec rake redmine:plugins:migrate NAME=redmine_feedback VERSION=0 RAILS_ENV=production
rm -rf plugins/redmine_feedback
```

После этого перезапустите Redmine.
