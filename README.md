# Redmine Feedback Plugin

Плагин для сбора обратной связи и оценок по задачам в Redmine.

## Возможности

- Сбор оценок (Хорошо, Нормально, Плохо) по задачам
- Возможность добавления комментариев к оценке
- **Перезапись оценки**: пользователи могут изменить свою предыдущую оценку
- Гибкая настройка поля для хранения оценок
- Использование макроса `{{feedback_link}}` для отображения ссылки на форму оценки

## Установка

1. Перейдите в директорию плагинов Redmine:
   ```bash
   cd /path/to/redmine/plugins
   ```

2. Клонируйте репозиторий плагина:
   ```bash
   git clone https://github.com/vlrazinov/redmine_feedback 
   ```

3. Установите зависимости:
   ```bash
   bundle install
   ```

4. Выполните миграцию базы данных:
   ```bash
   bundle exec rake redmine:plugins:migrate RAILS_ENV=production
   ```

5. Перезапустите Redmine:
   ```bash
   # Команда зависит от вашего способа запуска Redmine
   # Например, для Passenger:
   touch tmp/restart.txt
   # Или для systemd:
   sudo systemctl restart redmine
   ```

## Настройка

1. Зайдите в панель администрирования плагина в браузере:
   ```
   <URL REDMINE>/settings/plugin/redmine_feedback
   ```

2. Выберите пользовательское поле (Custom Field), которое будет использоваться для хранения оценок.

3. Нажмите кнопку **"Сохранить"**.

## Использование

Для отображения ссылки на форму оценки в задаче используйте макрос:

```
{{feedback_link}}
```

Макрос можно размещать в:
- Описании задачи
- Комментариях к задаче
- Шаблонах задач

### Отображение оценки с комментарием

Для отображения значения оценки в карточке задачи с всплывающей подсказкой (комментарием) при наведении, используйте хелпер метод `format_feedback_with_tooltip`:

```erb
<%= format_feedback_with_tooltip(@issue) %>
```

Этот метод:
- Получает значение оценки из настроенного пользовательского поля
- Находит связанный комментарий из таблицы `feedbacks`
- Если комментарий существует, оборачивает текст оценки в `<span>` с атрибутом `title`, который отображается как всплывающая подсказка при наведении курсора
- Добавляет визуальное оформление (пунктирное подчеркивание) для указания на наличие дополнительной информации

Пример использования в шаблонах Redmine (например, в `issues/show.rb` или через хуки):

```erb
<div class="feedback-rating">
  <strong>Оценка:</strong> <%= format_feedback_with_tooltip(@issue) %>
</div>
```

## Как это работает

1. Пользователь получает ссылку на форму оценки (через макрос или уведомление).
2. Открывает форму и выбирает оценку (Хорошо/Нормально/Плохо).
3. При необходимости добавляет комментарий.
4. Отправляет форму.
5. **Если оценка уже была оставлена**, система предложит перезаписать её новым значением.
